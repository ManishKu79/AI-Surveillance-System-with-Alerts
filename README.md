AI Surveillance System with Alerts

⚡ Real-time AI-powered surveillance system for face recognition, object detection, motion detection, and instant alerts


Overview

The AI Surveillance System is an intelligent security solution that automates monitoring using AI and computer vision.
It detects unknown faces, suspicious objects, and motion in real time and instantly sends alerts with captured evidence.

💡 Built to replace traditional CCTV systems with smart, automated surveillance


✨ Key Features

👤 Face Recognition (Known & Unknown)

🔍 Object Detection (YOLOv8)

⚠️ Motion Detection

📸 Automatic Evidence Capture

📲 Telegram Alerts (with images)

📧 Email Notifications

🗂️ Database Logging (SQLite)

⚡ Real-time Processing


🛠️ Tech Stack

🐍 Python

👁️ OpenCV

🧠 DeepFace

🎯 Ultralytics YOLO

🗄️ SQLite
🔢 NumPy
